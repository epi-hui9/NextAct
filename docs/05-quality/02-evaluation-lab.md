# Evaluation lab

Use this space for prompt / memory / output-policy experiments that are not yet product.

- Unit checks: `pnpm test` (`tests/integration/`).
- Mobile UX: `pnpm test:e2e`.
- Copy guard: `pnpm scan:banned-copy` (no “quiet”, no U+2014 em dash).

Capture lasting findings as ADRs or architecture notes, not as one-off chat transcripts.
